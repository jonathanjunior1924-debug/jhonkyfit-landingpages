## 2. Descripción del Proyecto
* **¿Qué hace el proyecto?** Es una "Landing Page" (página de aterrizaje) moderna e interactiva para una tienda virtual de suplementos deportivos y equipamiento de gimnasio.
* **¿Qué problema soluciona?** Brinda a los usuarios un catálogo digital rápido y atractivo para conocer los productos y beneficios de la marca antes de realizar una compra, mejorando la presencia online del negocio.
* **¿Para quién está pensado?** Atletas, fisicoculturistas, y entusiastas del fitness que buscan mejorar su rendimiento físico con productos de calidad.

---

## 3. Tecnologías Utilizadas
Para este proyecto se utilizó un entorno de desarrollo moderno basado en componentes:
* **React / Next.js:** Framework principal para la construcción de la interfaz.
* **Tailwind CSS:** Para los estilos y el diseño responsivo (colores oscuros y detalles naranjas).
* **Node.js / npm:** Para la gestión de paquetes y ejecución del entorno local.
* **Git y GitHub:** Para el control de versiones y almacenamiento del código.
* **Vercel:** Para el despliegue (hosting) del proyecto en producción.

---

## 4. Estructura del Proyecto
El proyecto sigue la estructura estándar de una aplicación moderna de Next.js:

* `/app`: Contiene la página principal y la configuración de rutas.
* `/components`: Partes reutilizables de la página como botones o tarjetas.
* `/public`: Imágenes, íconos y recursos estáticos que se ven en la web.
* `package.json`: Archivo que guarda la lista de dependencias (como React y Tailwind).
* `README.md`: Documentación oficial del proyecto.

---

## 5. Funcionalidades
1. **Hero Section (Sección Principal):** Un título impactante con botones de llamado a la acción.
2. **Navegación Superior (Navbar):** Menú con enlaces a diferentes secciones.
3. **Diseño Responsivo:** La página se adapta perfectamente a pantallas de celulares, tablets y computadoras.
4. **Catálogo de Productos:** Tarjetas visuales que muestran suplementos y equipamiento.
5. **Sección de Beneficios:** Área informativa que detalla la calidad y los resultados que ofrece la marca.

## 7. Cómo Ejecutar el Proyecto
Para correr este proyecto en modo de desarrollo local, sigue estos pasos:

1. **Clonar el repositorio:** `git clone https://github.com/jonathanjunior1924-debug/jhonkyfit-landingpages.git`
2. **Entrar a la carpeta del proyecto:**
   `cd jhonkyfit-landingpages`
3. **Instalar las dependencias:** `npm install`
4. **Encender el servidor local:**
   `npm run dev`
5. **Abrir en el navegador:** Ve a `http://localhost:3000` para ver la página.
